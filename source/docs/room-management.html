‚ú® Creating Rooms

Step-by-Step Room Creation

1

Access Creation Menu

  • Click "Create New Room" from main menu
  • Or use keyboard shortcut: Ctrl+N (Cmd+N on Mac)
2

Basic Room Information

  • Room Name: Choose a descriptive name (max 50 characters)
  • Description: Optional details about the room purpose
  • Room ID: Automatically generated 6-character code
3

Privacy Settings

üåê Public Room
  • Leave password field empty
  • Room appears in server listing
  • Anyone can join with room ID
üîí Private Room
  • Set a strong password (8+ characters)
  • Hidden from public listings
  • Share password with invited users

üö™ Joining and Leaving Rooms

Ways to Join Rooms

üÜî Direct Room ID Entry

  1. Click "Join Existing Room"
  2. Enter the 6-character Room ID
  3. Enter your display name for this room
  4. Enter password if required
  5. Click "Join Room"
Example Room ID: ABC123

üìã Room List Browser

  1. View "Available Rooms" on main menu
  2. Browse public rooms on the server
  3. Click on a room to see details
  4. Click "Join" to enter

üîó Direct Links

Room creators can generate shareable links:

https://your-server.com/join/ABC123
voicelink://join/ABC123 (desktop app)

üîñ Bookmarked Rooms

  1. Save frequently visited rooms as bookmarks
  2. Access from "My Bookmarks" menu
  3. One-click rejoining

Room States and Persistence

üü¢ Active Rooms

  • Currently has users present
  • All features fully available
  • Real-time communication active

üü° Dormant Rooms

  • No users currently present
  • Settings and history preserved
  • Can be rejoined anytime

üî¥ Expired Rooms

  • Duration limit reached
  • No longer accessible
  • Data automatically cleaned up

üíæ Persistent Rooms

  • Permanent or very long duration
  • Maintain user preferences
  • Remember 3D positions

Leaving and Returning to Rooms

üö™ Ways to Leave

  • Leave Button: Click door icon in room header
  • Keyboard Shortcut: Ctrl+Q or Cmd+Q
  • Close App: Automatically leaves all rooms
  • Network Disconnect: Automatic graceful disconnect

üîÑ Returning to Rooms

Same Session Return
  • Room ID remains in recent history
  • Rejoin with saved settings
  • Restore previous 3D position
Later Date Return
  • Use bookmarked rooms for easy access
  • Room must not have expired
  • Settings preserved for persistent rooms
Different Device Return
  • Sign in with same account
  • Bookmarks sync across devices
  • User settings carried over

üîß Common Issues and Solutions

‚ùå Cannot Create Room

Possible Causes:

  • Server at capacity (1000 user limit reached)
  • IP connection limit exceeded
  • Insufficient permissions
  • Network connectivity issues

Solutions:

  • Try again later when server load is lower
  • Check internet connection
  • Contact server administrator
  • Try a different server

üö´ Room Not Found

Possible Causes:

  • Incorrect room ID entered
  • Room has expired
  • Room is on different server
  • Private room with wrong password

Solutions:

  • Double-check room ID spelling
  • Verify you're on correct server
  • Contact room creator for current details
  • Check if room duration has expired

‚è∞ Room Expired Unexpectedly

Possible Causes:

  • Duration limit reached
  • Server maintenance
  • Host deleted room
  • System cleanup performed

Solutions:

  • Create new room with longer duration
  • Contact server administrator
  • Use lifetime rooms for permanent spaces
  • Set up automatic room recreation
