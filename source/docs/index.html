Current Baseline

  • Desktop-first release with macOS and Windows native clients.
  • Web/browser room surface should only be reopened when the browser client is ready.
  • Licensing, device activation, and connected account sync are now part of the normal account flow.
  • Server administration is role-gated and should only appear for the roles allowed to use it.
