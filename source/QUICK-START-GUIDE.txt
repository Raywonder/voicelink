FIRST TIME SETUP
================

1. LAUNCH THE APPLICATION
   - Double-click the VoiceLink Local app
   - Wait for the startup screen to load
   - The server will automatically start on port 3000

2. INITIAL CONFIGURATION
   - You are automatically assigned as Server Owner (ğŸ‘‘)
   - Access Admin Panel to configure your server

3. CREATE YOUR FIRST ROOM
   - Click "Create Room" from the main menu
   - Enter room name and settings
   - You'll automatically join after creation

INVITING OTHERS
===============

FOR APP USERS:
1. Share your Room ID (displayed in room)
2. Or use "Invite Users" in the room panel
3. They enter the Room ID to join

FOR WEB USERS (NO APP NEEDED):
1. Share link: http://[your-ip]:3000/room/[room-id]
2. They click link and join via browser
3. Works on all modern browsers
