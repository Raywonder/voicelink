How to connect to this VoiceLink server

Web Browser (Recommended)

Requirements:

  • Modern web browser (Chrome, Firefox, Safari, Edge)
  • Same Wi-Fi network as the server
  • JavaScript enabled
  1. Get the server URL from the host or copy it from the server settings. The URL looks like this:
    http://192.168.1.100:3000
  2. Open your web browser and paste the URL into the address bar.
  3. Allow microphone access when your browser asks for permission.
  4. Join or create a room by browsing available rooms or creating a new one.

Mobile Devices (iOS/Android)

  1. Open Safari (iOS) or Chrome (Android) on your mobile device.
  2. Navigate to the server URL that the host provided.
  3. Add to Home Screen (optional):
    • iOS Safari: Tap Share ‚Üí "Add to Home Screen"
    • Android Chrome: Menu ‚Üí "Add to Home Screen"
  4. Enable microphone when prompted by the browser.
  5. For iOS users: Tap the screen to activate audio if needed.

VoiceLink Desktop App

  1. Launch VoiceLink on your computer.
  2. Connect to external server:
    • Go to Settings ‚Üí Server Settings
    • Set "Server Mode" to "Connect to External"
    • Enter the server URL
    • Click "Connect"
  3. Join rooms normally through the app interface.

QR Code Connection

  1. Open camera app on your phone.
  2. Scan the QR code provided by the host.
  3. Tap the notification to open in your browser.
  4. Follow the web browser steps above.

üîß Troubleshooting

Can't connect to server:

  • Ensure you're on the same Wi-Fi network
  • Check if the server is running (ask the host)
  • Try refreshing the page
  • Disable VPN if active

No audio/microphone issues:

  • Check browser permissions (allow microphone)
  • Try refreshing and re-allowing permissions
  • On iOS: Tap screen to activate audio
  • Check device volume and microphone settings

Poor audio quality:

  • Move closer to Wi-Fi router
  • Close other bandwidth-heavy applications
  • Ask host to check server performance
