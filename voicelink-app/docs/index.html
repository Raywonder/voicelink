Common Tasks

Join Main or Community

Use the server list to choose the correct server, then activate a room row to open details, preview media, or join.

Check who is in a room

Open room details or join the room, then move to the user list and message area to confirm users, bots, and system messages.

Adjust audio safely

Open the room controls or user actions menu to change monitoring, mute, volume, or room background media without leaving the room.

Send diagnostics

Open Settings, choose diagnostics or bug reporting, and send a short description of the issue after reproducing it once.

Recommended Reading Order

  1. Read Getting Started if you are new to VoiceLink or switching devices.
  2. Read Using VoiceLink with VoiceOver if you rely on VoiceOver, keyboard navigation, or spoken status updates.
  3. Read Room Management if you create rooms, moderate rooms, or manage room media.
  4. Read Authentication if you need to sign in, reconnect, or work with more than one server.

Contact and Feedback

If the guide for your task does not match what the app says, send diagnostics from the app and include the server name, room name, and the exact label of the control that did not behave as expected.
