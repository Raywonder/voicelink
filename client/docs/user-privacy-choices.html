VoiceLink User Privacy Choices

Effective date: March 9, 2026

This page explains how VoiceLink users can request privacy-related actions and manage privacy choices available through the app, server settings, or support channels.

Your Choices

  • Sign out, disconnect linked providers, or switch servers.
  • Enable or disable supported notifications in client settings.
  • Manage microphone, camera, notification, photo, and file permissions in device settings.
  • Join, leave, mute, hide, or disable room participation as allowed by the server.

Requests You Can Make

  • Request account-data access or export.
  • Request correction of account information.
  • Request deletion or removal of account data, subject to server-owner retention or moderation requirements.
  • Request help understanding which integrations are enabled on the server you use.

Server Note

VoiceLink can be self-hosted or federated. A server operator may control retention, moderation logs, integrations, and some privacy settings for that server.

Support

For privacy requests, use https://voicelink.devinecreations.net/docs/contact.html.
