🔄 Auto-Updates

All native applications (v1.0+) include automatic update checking. The app will notify you when a new version is available and allow you to download and install updates directly from the app menu. You can check for updates manually at any time via the app's update settings.

🚀 Key Features

VoiceLink provides peer-to-peer voice communication with advanced 3D spatial audio, allowing you to hear participants positioned around you in virtual space. Features include push-to-talk, whisper mode, Jellyfin media synchronization, TTS announcements, and automatic port detection for seamless connectivity. Both local and federated server modes are supported for flexible deployment options.
