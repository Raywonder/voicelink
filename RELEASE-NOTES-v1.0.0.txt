KNOWN LIMITATIONS & FUTURE FEATURES
===================================

‚ö†Ô∏è WHAT MAY NOT WORK YET:
-------------------------

FEDERATION LIMITATIONS:
- Cross-server user authentication not fully tested
- Large-scale federation (10+ servers) may have performance issues
- Sub-server policy inheritance needs further testing
- Federation network discovery is manual only

OWNERSHIP TRANSFER NOTES:
- Transfer is one-way and irreversible
- Large data migrations (>1GB) may timeout
- Cross-platform transfers need verification
- External server compatibility depends on version matching

UI/UX AREAS FOR IMPROVEMENT:
- Some keyboard shortcuts not implemented in all screens
- Room preview feature limited in federated environments
- Mobile web interface needs optimization
- Offline mode indicators could be clearer

AUDIO/TECHNICAL:
- Spatial audio precision varies by hardware
- Some older browsers may have WebRTC limitations
- High-latency networks may affect real-time features
- Echo cancellation tuning needed for some audio setups

FUTURE DEVELOPMENT ROADMAP:
- Automatic federation discovery
- Mobile companion app
- Advanced spatial audio effects
- Room templates and automation
- Integration with external authentication systems
- Voice recording and playback features

INSTALLATION & UPGRADE
======================

FRESH INSTALLATION:
1. Download VoiceLink Local v1.0.0 DMG/installer
2. Install following platform-specific instructions
3. Launch application and complete initial setup
4. Configure user roles and server settings

UPGRADING FROM PREVIOUS VERSIONS:
1. Backup existing room and user data
2. Install v1.0.0 over existing installation
3. Launch app - settings will migrate automatically
4. Review new Admin Panel features
5. Configure user roles for existing users
6. Test federation features if needed
