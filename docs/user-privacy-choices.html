User Privacy Choices

Effective date: March 9, 2026

This page explains how VoiceLink users can request privacy-related actions and manage privacy choices that are available through the app, server settings, or support channels.

Your Choices

  • Account access: sign out, disconnect linked providers, or switch servers.
  • Notifications: enable or disable push or alert-style notifications in supported clients.
  • Microphone, camera, photo, file, and notification permissions: manage these in device settings.
  • Chat and room participation: join, leave, mute, hide, or disable room participation as allowed by the server.
  • Server-specific retention or moderation settings may affect what data is kept by a server operator.

Requests You Can Make

  • Request account-data access or export.
  • Request correction of account information.
  • Request deletion or removal of account data, subject to server-owner retention and moderation requirements.
  • Request help understanding which integrations are enabled on a server you use.

How to Contact Us

Use the main support page for privacy requests: Contact and Support.

Important Server Note

VoiceLink can be self-hosted or federated. A server operator may control data retention, moderation logs, integrations, and some privacy settings for that server. If you use a third-party or community server, privacy requests may need to be handled by that server operator in addition to Devine Creations.
