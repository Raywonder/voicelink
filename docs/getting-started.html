üåê Understanding VoiceLink's Decentralized Network

VoiceLink is a federated, decentralized platform. Unlike traditional voice chat services, VoiceLink connects users across multiple independent servers worldwide.

üñ•Ô∏è Choose Your Server

When connecting, you'll need to know which server you want to join. Each server may have different rooms, communities, and features.

üîó Federation

Federated servers can share rooms across the network. Rooms from other servers appear in your room list marked with their source.

üè† Self-Hosting

You can run your own VoiceLink server for complete control over your voice chat community and data.

Important: To connect, you need the URL of a VoiceLink server. Common options include:
  • Public Hub: https://voicelink.devinecreations.net (main public server)
  • Community Servers: URLs shared by communities you're part of
  • Self-Hosted: Your own server URL (e.g., https://voice.yourdomain.com)

üêß Server Setup

Host your own VoiceLink Local server for maximum control and privacy.

Minimum Requirements:

  • CPU: 2 cores, 2.0 GHz
  • RAM: 2GB (4GB recommended)
  • Storage: 1GB free space
  • Network: 10 Mbps upload/download
  • OS: Linux (Ubuntu 20.04+, CentOS 8+, Debian 11+)

Recommended for Large Deployments:

  • CPU: 8+ cores, 3.0+ GHz
  • RAM: 16GB+
  • Storage: SSD with 10GB+ free
  • Network: 100+ Mbps with low latency

Server Capacity:

Server Configuration Max Total Users Max Users Per Room Connections Per IP
Small (2 cores, 2GB RAM) 100 users 50 per room 2-3 per IP
Medium (4 cores, 8GB RAM) 500 users 250 per room 3-5 per IP

üè† Step 3: Join Your First Room

Join an Existing Room

Connect to a room someone else has created.

  1. Click "Join Existing Room" on the main menu
  2. Enter the Room ID (6-character code)
  3. Enter your Name for this room
  4. Enter the Password if the room is private
  5. Click "Join Room"

Create Your Own Room

Start a new room and invite others to join.

  1. Click "Create New Room" on the main menu
  2. Enter a Room Name (visible to participants)
  3. Set a Password (optional, for private rooms)
  4. Choose Max Users (2 to 5000)
  5. Select Room Duration:
    • 1 hour, 2 hours, 6 hours
    • 30 days, 69 days
    • Lifetime (permanent)
  6. Click "Create Room"

üéõÔ∏è Step 4: Essential Controls

Voice Controls

  • Mute/Unmute: Click microphone icon or press Space (if PTT enabled)
  • Deafen: Temporarily stop hearing others
  • Push-to-Talk: Hold Space to speak (if enabled)
  • Volume: Adjust your hearing volume with slider

User Interactions

  • Right-click users: Access context menu with options
  • Keyboard shortcuts: Shift+M (VoiceOver) or Applications key (Windows)
  • Direct messages: Send private messages to specific users
  • User volume: Adjust individual user volumes

3D Audio Controls

  • Drag users: Position others in 3D space
  • Your position: Move yourself around the virtual room
  • Room reverb: Choose from different acoustic environments
  • Spatial settings: Enable/disable 3D audio effects

Room Management

  • Leave room: Click the door icon to exit
  • Room settings: Access via settings gear (if you're the host)
  • User management: Kick, ban, or moderate users (host only)
  • Room info: View room details and participant list
