Build 85 focuses on room visibility and settings parity:

- Room chat is always available in the active room view, even when there are no messages yet.
- A dedicated Room Messages sheet mirrors the macOS room chat workflow more closely.
- People in Room and Room Messages remain reachable from Room Actions while connected.
- Profile and Settings sheets now include Done controls so users can get back to the room.
- macOS room history and live chat now preserve system-message payloads that use alternate keys such as body, text, author, or nested sender data.
- macOS and iOS both request up to 200 prior room messages for better room history after joining.

Please test joining a room from iOS and macOS, confirming People in Room shows other users, confirming prior user and system messages appear, opening Room Actions > Room Messages, opening Profile and Settings and returning with Done, and sending a new room message from both platforms.
