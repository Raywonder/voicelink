Escort Me

The Escort Me feature allows users to request a trusted companion to join their voice chat session for safety, guidance, or support.

Action Description

API Endpoints

Request Escort

POST /api/escort/request

Request an escort for a room session.

Cancel Request

POST /api/escort/cancel

Cancel an active escort request.

End Session

POST /api/escort/end

End an active escort session.

Get Available Escorts

GET /api/escort/available

Get list of available community escorts.

Check Request Status

GET /api/escort/status/:requestId

Check the status of an escort request.

Becoming an Escort

Community members can volunteer to be escorts and help other users.

Escort Requirements

  • Level 2 (Regular) membership or higher
  • Trust score of 70 or higher
  • At least 14 days of active membership
  • No active complaints or warnings
  • Completed escort training (when available)

Escort Benefits

  • Escort badge on profile
  • Contribution to community safety
  • Trust score bonus for completed escorts
  • Recognition in escort leaderboard
Safety First: Escorts should always prioritize the safety and comfort of the user they are escorting. Report any concerning behavior immediately.
