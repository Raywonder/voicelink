VoiceLink Native v1.0 - Video Scripts
=====================================

This folder contains scripts for generating promotional and tutorial videos
for VoiceLink Native.

VIDEO LIST
----------

01-intro.txt
    - Duration: 60-90 seconds
    - Overview of VoiceLink Native
    - Key features highlight
    - Call to action

02-membership-system.txt
    - Duration: 2-3 minutes
    - Three membership levels explained
    - Level = Device tier concept
    - Trust score system
    - Paid tier bonuses
    - Mastodon reputation bonuses

03-room-system.txt
    - Duration: 2-3 minutes
    - Guest rooms (temporary)
    - Permanent rooms (persistent)
    - OpenLink visitor rooms
    - Room capacity calculations
    - Device rotation modes
    - Server capacity (50-5000)

04-safety-escort.txt
    - Duration: 2 minutes
    - Escort Me feature explained
    - Request reasons
    - How to request an escort
    - Trusted escorts list
    - Becoming a community escort

05-wallet-ecrypto.txt
    - Duration: 2 minutes
    - eCrypto wallet integration
    - Embedded vs full app wallet
    - Wallet setup flow
    - Invite links with test coins
    - Using wallet for features

06-getting-started.txt
    - Duration: 3 minutes
    - Download and installation
    - First launch
    - Authentication (Mastodon/Email)
    - Server pairing
    - Creating and joining rooms
    - Auto-updates


PRODUCTION NOTES
----------------

Target Platforms:
- YouTube
- Website embed
- Social media clips

Style Guidelines:
- Clean, modern aesthetic matching app design
- Dark theme with purple/blue accent colors
- Smooth animations and transitions
- Clear narration with subtitles
- macOS-style interface mockups

Assets Needed:
- VoiceLink logo animation
- App icon
- Feature icons
- UI screenshots/recordings
- Badge graphics (levels, trust, paid tiers)
- Mastodon logo (with permission)
- eCrypto logo (with permission)

Audio:
- Professional voice-over
- Background music (royalty-free)
- UI sound effects (subtle)


HOSTING OPTIONS
---------------

Recommended: Host these scripts on the server
- Easy to update without app rebuild
- Can be fetched dynamically
- Version control via server

Alternative: Include in app bundle
- Available offline
- Larger app size
- Requires rebuild for updates
