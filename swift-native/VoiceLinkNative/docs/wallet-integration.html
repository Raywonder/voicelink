Wallet Types

VoiceLink supports two types of wallet integration:

üì±
Embedded Wallet

A wallet managed by VoiceLink for quick and easy setup.

  • Instant setup - no app required
  • Automatic test coins on creation
  • Tied to your authentication
  • Perfect for trying features
üîê
eCrypto App

Full wallet control with the dedicated eCrypto app.

  • Complete wallet control
  • Send & receive eCrypto
  • Use across multiple apps
  • Enhanced security features

Wallet Setup Flow

1
Authenticate
Sign in with Mastodon or Email to verify your identity. Your wallet will be linked to this authentication method.
2
Choose Wallet Type
Create an embedded wallet for quick start, or connect the eCrypto app for full control.
3
Receive Test Coins
New wallets automatically receive test coins from the faucet. Use these to try premium features!
4
Start Using
Pay for features, upgrade tiers, or transfer server ownership using your wallet.

Installing eCrypto App

Invite Link System

VoiceLink generates personalized invite links for eCrypto installation:

  • Personalized download link tied to your account
  • Bonus test coins included with installation
  • Server owners receive additional hosting credits
  • Share links to earn referral rewards
Test Coins: Test coins can be used to try all premium features. They work exactly like real eCrypto but are only valid within the VoiceLink ecosystem.

Wallet API Endpoints

Create Embedded Wallet

POST /api/wallet/create

Request Body

FieldTypeDescription
userIdStringAuthenticated user ID
authMethodString"mastodon" or "email"
appIdString"voicelink"
deviceIdStringDevice identifier
embeddedBooleantrue for embedded wallet

Request Test Coins

POST /api/faucet/request

Request Body

FieldTypeDescription
walletAddressStringWallet address
appIdString"voicelink"
reasonString"new_user" or "promotion"

Generate Invite Link

POST /api/invite/generate

Request Body

FieldTypeDescription
appIdString"voicelink"
hasServerBooleanUser owns a server
walletAddressStringReferrer wallet address
userIdStringUser ID

Response

FieldTypeDescription
inviteLinkStringPersonalized invite URL
bonusCoinsNumberCoins included with invite

Process Payment

POST /api/payment/process

Request Body

FieldTypeDescription
fromAddressStringPayer wallet address
amountNumberPayment amount
featureStringFeature being purchased
appIdString"voicelink"
useTestCoinsBooleanUse test coins if available

Get Balance

GET /api/wallet/balance?address={walletAddress}

Response

FieldTypeDescription
balanceNumberReal eCrypto balance
testBalanceNumberTest coin balance
